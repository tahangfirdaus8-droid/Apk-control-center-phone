# Remote Control APK

Aplikasi remote control sederhana untuk Android.

## Fitur
- 🔒 Lock Device
- 🔓 Unlock Device
- 💡 Flash Blink
- 🖼️ Screenshot

## Cara Build
1. Fork repository ini
2. Buka tab Actions
3. Klik "Build APK" → "Run workflow"
4. Download hasil dari Artifacts
